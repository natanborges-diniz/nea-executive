# Essilor x Hoya x Zeiss — Comparação Inicial

## Padrão comum confirmado
Os 3 fornecedores compartilham a mesma estrutura problemática:
- catálogo técnico + comercial + preço no mesmo documento
- nomes proprietários dificultando comparação
- claims de marketing misturados com benefícios técnicos
- disponibilidade e regras de combinação espalhadas

## Força principal de cada fornecedor
### Essilor
- força em personalização, adaptação e marcas fortes
- Varilux, Crizal, Eyezen, Stellest, Transitions

### Hoya
- força em precisão, binocularidade e materiais/tecnologias proprietárias
- Hoyalux, Nulux, Hilux, MiYOSMART, Sensity, Hi-Vision

### Zeiss
- força em personalização, IA, BlueGuard, UVProtect e progressivos premium
- SmartLife, DuraVision, PhotoFusion X, MyoCare, DriveSafe, OfficeLens

## Comparabilidade real hoje
### Já comparável
- categoria clínica
- posicionamento comercial aproximado
- materiais básicos por índice
- famílias principais
- benefícios explícitos gerais

### Ainda não comparável com segurança
- equivalência fina entre tecnologias
- preço por combinação
- disponibilidade completa
- equivalência exata de materiais proprietários
- equivalência 1:1 entre famílias premium

## Decisão
O produto deve comparar primeiro por:
1. categoria clínica
2. posicionamento comercial
3. benefícios percebidos
4. apoio visual

E só depois por:
5. material
6. tratamento
7. preço
8. disponibilidade
