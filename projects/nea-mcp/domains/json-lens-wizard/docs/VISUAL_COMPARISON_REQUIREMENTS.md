# Visual Comparison Requirements

## Objetivo
Garantir que a ferramenta compare lentes e tecnologias também por ilustração visual e não apenas por texto.

## Elementos visuais prioritários
1. campo visual
2. zonas de leitura / intermediário / longe
3. tratamento antirreflexo
4. blue light / UV
5. fotocromia
6. espessura / estética do material
7. conforto / adaptação
8. progressão de valor entre famílias

## Tipos de interface sugeridos
- cards comparativos com ícones e mini-ilustrações
- diagramas simples de campo visual
- barras comparativas de benefícios
- overlays visuais para tratamentos
- before/after conceitual
- escala visual de posicionamento comercial

## Regra
Visual não substitui dado técnico.
Visual deve traduzir tecnologia em entendimento rápido de venda.

## Aplicação comercial
A camada visual deve ajudar o vendedor a responder rapidamente:
- o que melhora
- por que custa mais
- qual ganho o cliente percebe
- qual upgrade faz mais sentido
