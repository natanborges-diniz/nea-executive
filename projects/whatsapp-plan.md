# WhatsApp Integration Plan

## Objetivo
Definir a melhor arquitetura para a integração operacional do WhatsApp do Natan com a Nea.

## Pergunta central
Como permitir que a Nea ajude na operação do WhatsApp com segurança, governança e utilidade real?

## Opções a avaliar
1. suporte nativo do OpenClaw para WhatsApp
2. integração via API externa (ex.: Evolution)
3. arquitetura híbrida com separação entre canal e camada operacional

## Critérios de decisão
- segurança
- estabilidade
- esforço de manutenção
- visibilidade operacional
- risco de bloqueio
- custo
- aderência ao fluxo real do Natan

## Próximos passos
1. revisar arquitetura suportada pelo OpenClaw
2. comparar com ponte externa
3. recomendar caminho principal
4. definir plano de implantação
